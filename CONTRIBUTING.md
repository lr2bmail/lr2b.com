# Contributing

This is a small static website repository.

## Workflow

1. Create a branch from `main`.
2. Keep changes focused.
3. Verify the page on desktop and mobile.
4. Open a pull request with a short summary and screenshots when visual changes are included.

## Quality Bar

- Keep visible copy short.
- Avoid heavy dependencies.
- Do not add generated images unless they serve a clear purpose.
- Preserve SEO metadata, sitemap, robots file, and structured data.
