# LR2B

Minimal static landing page for LR2B.

## Files

- `index.html`
- `script.js`
- `robots.txt`
- `sitemap.xml`
- `assets/`

## Cloudflare Pages

Use these settings:

- Framework preset: `None`
- Build command: leave empty
- Build output directory: `/`

## Local Preview

Open `index.html` directly, or run:

```bash
python -m http.server 8080
```
